How to use this folder:

1. Make sure you have Python 3.6+ installed.
2. Install required packages in your terminal:
   pip install pandas mlxtend
3. Open your terminal or VS Code terminal in this folder.
4. Run the script:
   python product_recommender.py
5. When prompted, enter a product name (e.g., milk, bread, eggs).
6. You will see product recommendations based on association rules.

Note: The product names are not case-sensitive.
